# Personal Portfolio Website

This project is a **Personal Portfolio Website** developed using **HTML and CSS**. The purpose of this website is to introduce myself, showcase my skills, and present the projects I have worked on during my web development learning journey.

The portfolio contains different sections such as **Home, About, Skills, Projects, and Contact**, which help visitors easily understand my background and abilities. It also includes a profile section with a short introduction and information about my technical skills.

## Features

* Navigation bar for easy section access
* Hero section with profile image and introduction
* About section describing personal background
* Skills section highlighting technical abilities
* Projects section showcasing completed work
* Contact section with basic contact information
* Clean and simple user interface

## Technologies Used

* HTML5
* CSS3

## Purpose of the Project

The main goal of this project is to practice **front-end web development basics** and learn how to structure and design a personal portfolio website using HTML and CSS.
